# Changelog

All notable changes to the PPG ML Service will be documented in this file.

## [1.0.0] - 2024-01-XX

### Added
- Initial release of PPG ML Service
- PaPaGei model integration for PPG signal analysis
- FastAPI backend with REST API endpoints
- Preprocessing pipeline for PPG signals
- Test suite and verification scripts
- Docker and Cloud Run deployment configurations
- Windows and Linux/Mac setup scripts
- Model download utility
- Integration verification tools
- Comparison tool (traditional vs ML processing)
- Comprehensive documentation

### Features
- Heart rate detection (40-200 BPM)
- Heart rate variability (HRV) calculation
- Respiratory rate estimation
- Signal quality assessment
- Confidence scoring
- Automatic fallback to traditional processing

### Documentation
- Quick start guide
- Deployment guide
- Development workflow guide
- Integration documentation
- API documentation
